How-To-Contribute
=================

In this section you will find instructions for contributing to Cata-Log project.

To get started and set up your local copy of the git repo, see :doc:`the quickstart <contributing-quickstart>`.

If you want to add a new provider either as a builtin or as a plugin, you should check out :doc:`the how-to guides <how-to>`.

To get involved with consolidation and optimization, please see :doc:`the ToDo list <todo>` for ideas on what to do.
